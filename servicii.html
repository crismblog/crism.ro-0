Instalare și depanare distribuții GNU/Linux

Ofer servicii comerciale de instalare și depanare GNU/Linux pentru clienții care țin la libertatea lor și aleg programele libere.

  • instalare sistem de operare GNU/Linux.
  • asistență pentru clienți care folosesc sistemul de operare GNU/Linux.
  • facturile sunt emise de PFA.

Sistem de operare

Sistemul de operare GNU+Linux pe care le recomand:

  • Trisquel - este un sistem de operare complet liber, care nu are nici un fel de program neliber instalat (gen - adobe flash, java, skype etc.).
  • Trisquel Sugar - este un sistem de operare complet liber, creat special pentru sistemul de învățământ.
  • Mint - este un sistem de operare parțial liber și este orientat spre utilizatorul cotidian.
  • Kogaion - este un sistem de operare parțial liber dezvoltat de o echipă românească.

Aceste sisteme de operare vin cu programe libere utile și uzuale instalate implicit, precum:

  • Abrowser, Firefox, Chromium sunt navigatoare pentru internet.
  • LibreOffice este o suită de aplicații pentru birou.
  • Clientul pentru poștă electronică, Evolution sau Thunderbird.
  • Editorul pentru grafică, GIMP.
  • Pidgin, popularul client de chat.
  • Playerul de filme, Totem sau VLC.
